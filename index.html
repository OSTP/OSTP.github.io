The Open Source Textbook Project

Making it easy to learn and educate

Welcome to the Open Source Textbook Project

OSTP aims to make it truly easy to make textbooks open source. You can download and use a textbook for a class, upload your own textbook for others to use, or modify an existing textbook. If you are familiar with the "git work flow", we aim to bring that workflow to the development of textbooks in an open source environment.
